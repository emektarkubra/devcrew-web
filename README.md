# Required Installations

- [Node.js](https://nodejs.org/)
- [Docker](https://www.docker.com/)
- [Visual Studio Code](https://code.visualstudio.com/)

# Running the Project

1. Clone the project:
    ```bash
    git clone https://github.com/emektarkubra/base-code-react.git
    ```

2. Navigate to the project directory:
    ```bash
    cd base-code-react
    ```

3. nstall the dependencies:
    ```bash
    npm install
    ```


4. Open the project with Visual Studio Code:
    ```bash
    code .
    ```
5. Update the Docker container ID in the start.sh file. (Only applicable for macOS; different files should be added for different versions)

6. Start the project:
    ```bash
    npm run dev
    ```
